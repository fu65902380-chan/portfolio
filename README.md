# fusiyang.com redesign

This is a lightweight static portfolio rebuild designed to replace a builder-based personal site with a lower-cost setup.

## Files
- `index.html` — main one-page site

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Set the source to **Deploy from a branch**.
5. Choose the `main` branch and `/root`.
6. Add a file named `CNAME` containing:
   `fusiyang.com`
7. In your domain DNS, point `@` to GitHub Pages A records and `www` to your GitHub Pages hostname.

## What to replace first
- Hero background image
- Project card titles and links
- Stills gallery images
- Social profile URLs
- Contact details if needed

## Notes
- The current version uses some placeholder stock imagery so the layout feels complete immediately.
- You can later split this into multiple pages if you want separate Works / Vlog / Stills pages.
